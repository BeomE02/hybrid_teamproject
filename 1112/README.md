# 제작 프롬프트
1. "토스 브랜드 가이드라인을 참고해서 CSS 변수 시스템을 만들어줘. 색상은 --toss-blue: #0064FF를 메인으로, Pretendard 폰트를 사용하고, 모바일 우선 반응형으로 설계해줘."
2. "코르도바와 GitHub Pages 둘 다 호환되는 index.html 기본 구조를 만들어줘. viewport, CSP 정책, PWA manifest 링크를 포함해줘."
3. "토스 스타일의 헤더 네비게이션을 만들어줘. 데스크톱은 가로 메뉴, 모바일은 햄버거 메뉴로 변환되게 해줘. 스크롤시 그림자 효과도 추가해줘."
4. "히어로 섹션을 만들어줘. '금융의 모든 것, 토스에서 쉽고 간편하게' 타이틀과 CTA 버튼을 포함하고, 배경 그라데이션을 적용해줘."
5. "프로젝트 폴더 구조를 설정해줘. /css, /js, /assets, /calculators 폴더로 구성하고, 각 폴더에 필요한 파일들을 나열해줘."
Phase 2: 금융 계산기 UI (Day 2)
6. "5개 계산기를 전환할 수 있는 탭 네비게이션을 만들어줘. 환율/대출/적금/연말정산/취득세 탭을 만들고, 활성 탭은 토스 블루 색상으로 표시해줘."
7. "계산기 공통 input 컴포넌트를 만들어줘. 숫자 입력시 천 단위 콤마, 포커스 애니메이션, 툴팁 도움말 기능을 포함해줘."
8. "환율 계산기 UI를 만들어줘. KRW 입력 필드, 8개국 통화 선택 드롭다운, 실시간 환율 표시 영역을 포함해줘."
9. "대출 상환 계산기 UI를 만들어줘. 대출금액/이자율/기간 입력, 원리금균등/원금균등 라디오 버튼을 포함해줘."
10. "계산 결과를 표시하는 공통 result-container 컴포넌트를 만들어줘. 슬라이드업 애니메이션과 공유/복사 버튼을 포함해줘."
Phase 3: 계산기 로직 구현 (Day 3)
11. "환율 계산 함수를 만들어줘. USD, EUR, JPY 등 8개 통화 대상, 1.75% 수수료 적용, 결과를 통화 기호와 함께 표시해줘."
12. "대출 상환 계산 함수를 만들어줘. 원리금균등과 원금균등 두 방식 모두 지원하고, 월 상환액과 총 이자를 계산해줘."
13. "적금 만기 계산 함수를 만들어줘. 복리 계산, 세금 15.4% 선택 적용, 프로그레스바로 수익률 시각화 기능을 포함해줘."
14. "연말정산 환급금 계산 함수를 만들어줘. 급여/부양가족/카드/의료비/교육비를 입력받아 예상 환급액을 계산해줘."
15. "부동산 취득세 계산 함수를 만들어줘. 주택 유형별/지역별 세율 적용, 지방교육세와 농어촌특별세까지 계산해줘."
Phase 4: 사용자 경험 개선 (Day 4)
16. "입력값 유효성 검사 시스템을 만들어줘. 빈 값, 음수, 범위 초과시 에러 메시지를 표시하고 포커스를 이동시켜줘."
17. "토스트 알림 시스템을 만들어줘. success/error/info 타입별 색상, 3초 후 자동 사라짐, 하단 중앙 표시를 구현해줘."
18. "계산 결과 공유 기능을 만들어줘. Web Share API 사용, 폴백으로 클립보드 복사, 카카오톡 공유 버튼도 추가해줘."
19. "로딩 스피너와 스켈레톤 UI를 만들어줘. 계산 중일 때 표시되는 로딩 상태, 토스 스타일의 shimmer 효과를 적용해줘."
20. "localStorage를 활용한 계산 히스토리 기능을 만들어줘. 최근 10개 계산 내역 저장, 재계산 기능을 포함해줘."
Phase 5: 모바일 최적화 (Day 5)
21. "터치 제스처를 구현해줘. 탭 간 스와이프 전환, pull-to-refresh, 길게 눌러 복사 기능을 추가해줘."
22. "모바일 키패드 최적화를 해줘. 숫자 입력시 넘버패드 자동 표시, 다음 입력 필드로 자동 이동 기능을 구현해줘."
23. "바텀 시트 모달을 만들어줘. 모바일에서 계산 결과를 바텀시트로 표시, 드래그로 높이 조절 가능하게 해줘."
24. "오프라인 모드를 구현해줘. Service Worker로 캐싱, 오프라인시 기본 환율 사용, 온라인 복귀시 자동 업데이트를 구현해줘."
25. "모바일 성능 최적화를 해줘. 이미지 lazy loading, CSS will-change 속성, requestAnimationFrame 사용을 적용해줘."
Phase 6: 배포 준비 (Day 6)
26. "PWA manifest.json을 작성해줘. 앱 아이콘, 스플래시 스크린, 테마 색상, 오프라인 페이지를 설정해줘."
27. "코르도바 config.xml을 작성해줘. 앱 이름, 버전, 권한 설정, iOS/Android 플랫폼별 설정을 포함해줘."
28. "GitHub Actions 워크플로우를 작성해줘. main 브랜치 푸시시 자동 빌드, GitHub Pages 배포, 빌드 실패시 디스코드 알림을 설정해줘."
29. "성능 모니터링 코드를 추가해줘. Google Analytics 4, 페이지 로드 시간, 계산기별 사용 통계를 수집하는 코드를 만들어줘."
30. "최종 테스트 체크리스트를 만들어줘. 브라우저별 호환성, 모바일 기기별 테스트, 접근성 검사 항목을 포함해줘."
